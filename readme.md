# dcinside.com gallery crawler
# 디씨인사이드 갤러리 크롤러

1. 목적
디씨 인사이드에 올라오는 특정 갤러리의 글을 수집하여
형태소나 글의 내용을 비교하고
가장 인기있는 단어나 요소들을 찾아본다.
최종적으로는 정리된 요소를 통하여 인터랙티브 서버를 띄우는 것이 목표.

상시 크롤링은 자원이 많이 낭비되고
가장 저렴한 서버를 이용중이므로
크롤러 파트와 인터랙티브 파트를 따로 분류하여 만든다.

2. 사용된 기술
헤드리스 크롬, puppeteer, node.js
